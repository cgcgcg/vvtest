
MOUSE:
- Left button: View task details ( If clicking on graph backgound,
    general info will apear )
- Double click(left button): Expand/Collapse child tree
- Middle button: Scroll

CONTROLS:
- Edit the source(s) of data with the "Edit Log Sources" button
- Update graph from log files with the "Update" button
- Change view modes with "24h View"/"48h View" buttons
- Go to specific date using the three text boxes
- View details in light grey panel
- Search tasks and their attributes with the long text box
- Toggle debug mode to view more task info or add more
  search options
